# Durability Reference Table — Core

Use this table as the primary scientific lookup for durability-/planning-skill explanations.

## Curation Contract

- Include only higher-authority sources: peer-reviewed studies, reviews, or central books with clear methodological relevance.
- Prefer DOI and/or PubMed identifiers when they are already available from local sources.
- If no stable DOI or PubMed identifier is available in the local seed source, record a stable alternative or `none`.
- `key_takeaways` must summarize the source briefly; they must not invent new rules.
- `authority_limit` is mandatory. Even core references remain evidence support, not direct plan authority.
