## Core Concepts

- Durability is the ability to preserve performance and physiological control as work accumulates over time.
- Late-session performance should be interpreted relative to energetic preload, not as if the athlete were fresh.
- Fresh peak markers such as VO2max, FTP, or threshold values do not fully describe how well an athlete resists prolonged fatigue.

## Key Takeaways

- Defines durability as performance preservation under prolonged work.
- Separates durability from fresh peak metrics.
- Frames durability as a distinct profiling target, not a side-note.

## Important Findings

- Argues that endurance profiling should include pre-fatigued assessment, not only rested testing.
- Explains that the same fresh physiological profile can produce different late-session outcomes once workload accumulates.
- Connects durability with pacing stability, drift behavior, and resistance to multi-system decline over long sessions.

## Practical Implications for RPS

- Use durability as a season/phase planning lens when the athlete targets long events, not only as a retrospective metric.
- Do not infer long-event readiness from fresh FTP/VO2max values alone.
- Interpret late-session fade, drift, and execution stability as meaningful planning signals.
