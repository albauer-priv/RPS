## Key Takeaways

- Prior accumulated work changes later output.
- Supports preload-aware interpretation of late-session performance.
- Highlights that not all fatigue exposure is interchangeable.

## Important Findings

- Supports the idea that accumulated workload and intensity history both shape what an athlete can do later.
- Provides direct rationale for treating pre-fatigue and finish-quality work as meaningful planning variables.
- Suggests that late-session decrements should not be evaluated without context on the work that preceded them.

## Practical Implications for RPS

- In RPS, use this as support for durability workouts that place meaningful work after prior load rather than in a fresh state.
