# Warmup, Activation, Add-On, Cooldown Rules

Warmup:
- always required
- short, structured, low-risk
- no hidden sustained threshold/VO2 work

Activation:
- required for VO2max, Threshold, and Sweet Spot
- optional for Tempo
- must not be used to patch an illegal warmup

Add-On:
- optional aerobic extension only
- must not change the workout's governing intensity classification

Cooldown:
- always required
- simple, low intensity, and explicit
