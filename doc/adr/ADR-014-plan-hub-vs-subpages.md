# ADR-014: Plan Hub vs. Separate Plan Subpages

**Status:** Accepted  
**Date:** 2026-02-01  

## Context

The UI can either be a single guided Plan Hub or split by artifact stage. We need a consistent structure that stays predictable for athletes.

## Decision

Keep Plan Hub as the summary/orchestration layer and retain separate Plan subpages (Season/Phase/Week/Workouts) for depth.

## Consequences

- Plan Hub provides readiness + execution status.
- Subpages handle detailed review and manual actions.

## Exceptions

None.
